## Installation

To install the library and command line program, use the following:

```
go get -u github.com/mlabouardy/butler
```

## Available Commands

### Jobs Management

```
$ butler jobs export --server http://localhost:8080 --username admin --password admin
```

```
$ butler jobs import --server http://localhost:8080 --username admin --password admin
```

### Plugins Management

```
$ butler plugins export --server http://localhost:8080 --username admin --password admin
```

```
$ butler plugins import --server http://localhost:8080 --username admin --password admin
```
